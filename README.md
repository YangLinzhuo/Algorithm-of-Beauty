# Algorithm-of-Beauty


- 《算法之美——隐匿在数据结构背后的语言》本书以现代计算机常用的十八种数据结构为线索，结合C++中的STL编程实践
- 详细介绍了四大算法设计思想(贪心法、动态规划、分治法、回溯法)、二十大经典问题和四十二个重要算法。
- 具体涉及的数据结构类型包括：数组、字符串、链表(单向链表、单向循环链表、双向循环链表)、栈、队列、树(二叉树、哈夫曼树、堆)
- 森林、搜索树(二叉搜素树、AVL树、红黑树、Trie树)、图、集合、字典和并查集。
